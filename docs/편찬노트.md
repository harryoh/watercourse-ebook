# 편찬 노트 — 원고별 manuscript.md 편찬 기록

각 책의 `manuscript.md`를 **무엇에서 / 어떻게** 만들었는지 기록한다. 다음에 다시 만들거나
새 책을 추가할 때 같은 방법을 재사용하기 위한 문서다.

## 두 가지 추출 경로
- **표준 추출기**: `shared/lib/extract_hwp.py` — HWP(또는 .docx) → manuscript.md.
  대부분의 강해집은 이걸로 됨. 차례(점선/밑줄/공백/`[N]`/`│차례│`)와 본문 `번호+제목+성경구절`
  구조를 인식하고, 러닝헤더·꼬리말·페이지번호 노이즈를 제거한다.
  - 일반 실행:  `python3 shared/lib/extract_hwp.py books/<슬러그>`
  - 메타데이터/표지 자동초안까지: `python3 shared/lib/mkbook.py <슬러그> <원본.hwp> <성경책> <권번호> <배경색> <강조색>`
- **책별 커스텀 빌더**: 구조가 특이한 책은 `books/<슬러그>/build_manuscript.py` 로 따로 만든다.

> ⚠ `manuscript.md`는 손편집 마스터다. 추출기는 기존 파일을 덮어쓰지 않고 `manuscript.regen.md`로 저장한다(`--force`로 강제).

## 새 책 추가 시(요약)
1. 표준형이면: `python3 shared/lib/extract_hwp.py books/<슬러그>` (book.env에 HWP_SOURCE·BIBLE_REF 먼저 설정)
2. 안 잡히면 `python3 shared/lib/dump.py 원본` 으로 원문을 보고, 가장 비슷한 책의 `build_manuscript.py`를 복제해 수정
3. 추출 후 `review.py`로 점검 → 사람 검토 → `book.env` `REVIEWED="yes"` → 빌드
