## 원본 & 추출
- **원본**: `original/내가 사랑하는 자야2.hwp`
- **방법**: 커스텀 빌더(특이 구조)
- **특이사항**: 기도 묵상집(ㄱㄴㄷ, 119편). 시(詩)형 본문. 본문 사진 5장+교회안내 위치 복원.

## 재생성 / 빌드
```bash
python3 "books/08. 내가 사랑하는 자야(기도문)/build_manuscript.py"   # 커스텀 빌더(소스 위치 의존)
python3 shared/lib/review.py "books/08. 내가 사랑하는 자야(기도문)"
./build.sh "books/08. 내가 사랑하는 자야(기도문)"
```

## 표지(앞표지)

- **원본 표지**: `표지/내가사랑하는자야_표지_1최종.ai` — 앞·뒤표지+책등(+날개) **펼침**(표지 AI(=PDF 호환)).
- **추출**: 300dpi 변환 후 **앞표지 영역만** 잘라 폭 1400px JPEG(품질 90)로 저장. 책등·뒤표지·날개 제외.
- **앞표지 영역(비율)**: 좌 50.5% ~ 우 79.5%, 상 4.5% ~ 하 93.5%  → `book.env`의 `COVER_CROP`
- **재현**(book.env의 COVER_SRC/COVER_CROP 사용):
```bash
python3 shared/lib/extract_cover.py "books/08. 내가 사랑하는 자야(기도문)" --src-root "<원본 표지가 있는 폴더>"
```
