# 모든 사명을 사랑으로 감당하라(고린도전서) — 제작 노트

이 책의 EPUB을 만들 때 참고. (저장소 루트에서 실행, 폴더명에 공백이 있으니 따옴표로 감쌀 것)

## 개요
- **제목**: 모든 사명을 사랑으로 감당하라 / **저자**: 조춘숙 목사
- **성경/주제**: 고린도전서 (강해집 2 · 15~27강)
- **장 수**: 13 (15~27강. 1~14강은 별권 "강해집 1")
- **판/발행**: 제1판 · 2014-02 · 전자책 v1.0.0
- **검토(REVIEWED)**: (사람 검토 후 yes 표시)

## 원본 & 추출 (⚠ 비표준 — 전용 추출기 사용)
- **원본**: `original/고린도전서합본.hwp` (표지 `※`동영상 안내가 구 URL이라 book.env VIDEO로 표준화)
- **방법**: **전용 추출기** `tools/extract_06.py` (표준 `extract_hwp.py`로는 안 됨)
- **비표준 사유와 보정**:
  - 목차 헤더가 `C.O.N.T.E.N.T.S` → 표준 차례 파서가 못 잡음 → TOC를 스크립트에 수동 주입.
  - 장 번호가 1이 아니라 **15~27** (강해집 2). review.py의 "장 번호 불연속" 경고는 **정상**.
  - 장 도입 성경구절이 `… 절 말씀` 접미사를 가짐 → opener(접두 일치)로 인식.
  - 도입 성경/인용이 여러 문단(절 번호로 시작)으로 쪼개짐 → 연속 절 문단을 한 인용으로 **병합**.
  - 머리말에 러닝헤더(`이 책을 읽는 분들께`, `고린도전서강해집 2`)가 새어 들어옴 → 제거.
- **검증**: 원본 본문 문단 전수 커버리지 확인(러닝헤더 외 누락 0), 인용박스 과병합 0.

## 재생성 / 빌드
```bash
python3 "books/06. 모든 사명을 사랑으로 감당하라(고린도전서)/tools/extract_06.py"   # HWP → manuscript.md (덮어씀)
python3 shared/lib/review.py "books/06. 모든 사명을 사랑으로 감당하라(고린도전서)"
./build.sh "books/06. 모든 사명을 사랑으로 감당하라(고린도전서)"
```
> 주의: 전용 추출기는 표준기와 달리 기존 manuscript.md를 **덮어쓴다**. 수동 교정 후 재추출 시 백업할 것.

## 표지
- 원본 표지 파일 없음 → **표지 없이** 빌드. (추후 표지 입수 시 `book.env`에 COVER_SRC/COVER_CROP 추가 후 `extract_cover.py`)
