
## 표지(앞표지)
- **원본 표지**: `빌립보서.png` — 앞·뒤표지+책등+날개 **펼침** 이미지(5348×2860).
- **추출**: 앞표지 영역만 비율로 잘라 폭 1400px JPEG(품질 90)로 저장. 책등·뒤표지·날개 제외.
- **앞표지 영역(비율)**: 좌 50.5% ~ 우 79.5%, 상 4.5% ~ 하 93.5% → `book.env`의 `COVER_CROP`
- **재현**(book.env의 COVER_SRC/COVER_CROP 사용):
```bash
python3 shared/lib/extract_cover.py "books/14. 주 안에서 같은 마음을 품으라(빌립보서)" --src-root "<빌립보서.png 가 있는 폴더>"
```
